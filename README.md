# Universe-clicker-simulator

Кликер про космос, в котором идёт эволюция астероида в другие, более массивные небесные тела.

## 🚀 Особенности
- **Прокачка уровня гравитации**: увеличивайте силу клика, чтобы быстрее копить очки гравитации.
- **Визуальная эволюция**: ваш астероид меняет внешний вид на 1,2,3,4 и 5 уровнях.
- **Автокликер**: Тратя очки вы покупаете автокликер, который даёт определенное кол-во очков за некоторое время.
- **Прокачка Автокликера**: Тратя очки, покупаете прокачку времени действия автокликера и количество получаемых очков в секунду
